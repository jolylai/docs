# Docs

This is the repository for compiling my note.


## 👻 Start
```bash
# clone repository
git clone https://github.com/jolylai/docs.git

# change direction to docs
cd docs

# install dependance
yarn

# start
yarn run start
```

if have a trouble on sharp when install dependance(windows platform)
```bash
npm install -g  windows-build-tools@1.3.2
```
